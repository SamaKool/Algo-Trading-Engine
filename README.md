# Vectorized-Algorithmic-Trading-Engine-or-Dynamic-Moving-Average-Backtester
A robust, object-oriented backtesting engine capable of simulating technical analysis strategies on historical financial data. It features a vectorized execution model for high-performance testing and includes a parameter optimizer to identify alpha-generating signal combinations.
